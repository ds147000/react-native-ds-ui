# react-native-mb-ui

mb-ui,react-native-ui,具有反馈组件和基础组件的ui库，

## Installation

```sh
npm install react-native-mb-ui
```

## Usage

```js
import MbUi from "react-native-mb-ui";

// ...

const result = await MbUi.multiply(3, 7);
```
